# 项目文档

## 项目概述
这是一个示例项目,用于测试文档更新技能。

## 架构
- 前端框架: React
- 状态管理: 无
- 构建工具: Vite

## 组件
所有组件位于 `src/components/` 目录。

### Button 组件
基础按钮组件,支持点击事件、禁用状态和多种样式变体 (primary/secondary/danger)。详见 [patterns/button-component.md](./patterns/button-component.md)。

## 工具函数
工具函数位于 `src/utils/` 目录。

## 开发约定
- 使用 TypeScript
- 组件使用函数式写法
- 导出使用 default export
